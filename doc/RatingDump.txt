- Alle Spiele müssen aktuell sein ( Einfach alle eigenen Spiele löschen und via archiv download neu laden )
- Alle HRF's importiert sein
- Subskills müssen berechnet sein

- Dump dauert ( P2,8 GHZ -> 20 min )
- 2te Phase vom dump muss sich an den ht server connecten ( derby-check ), vorsicht vor Timeout falls vor dem export ein hrf oder so gezogen wurde!

Die xmldatei verwendet die HO-Konstanten für Werte.
Siehe ISpieler für Erfahrung und ILineup für System, usw...
