Changelist for HO! 1.430
========================

This is a full release based on changes during betas 1.427 to 1.429

Major news during this period is:

- A 553 based lineup interface
- Authentication through oauth
- Updated training values
- Rating prediction is again correctly done through the default prediction set

For full details on changes since 1.426, see the changelog.txt for 1.427 to 1.429
